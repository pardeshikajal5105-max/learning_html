Basics of HTML

**Basics of HTML** HTML (HyperText Markup Language) is the standard markup language used to create and design web pages. It provides the basic structure of a website by organizing content such as headings, paragraphs, images, links, tables, lists, and forms. HTML uses **tags**, which are enclosed in angle brackets (`< >`), to define different elements of a webpage. Most HTML tags have an opening tag and a closing tag, while some, like `
` and ``, are self-closing. Every HTML document begins with the `` declaration, which tells the browser that the document is written in HTML5. The root element of the document is the `` tag, which contains two main sections: the `` and the ``. The `` section stores information about the webpage, such as the title, character encoding, metadata, and links to external CSS or JavaScript files. The `` section contains all the visible content that appears on the webpage, including text, images, buttons, links, and other elements. HTML elements can also have **attributes**, which provide additional information about an element. For example, the `href` attribute specifies the destination of a hyperlink, while the `src` attribute specifies the source of an image. Attributes are always written inside the opening tag. HTML is simple to learn, platform-independent, and supported by all modern web browsers. However, HTML alone is used only to define the structure of a webpage. To improve the appearance of a webpage, **CSS (Cascading Style Sheets)** is used, and to add interactivity and dynamic behavior, **JavaScript** is used. WEB DEVELOPMENT

Web development is the process of designing, developing and maintaining websites and web apps.[1] Web development encompasses several different fields, most commonly referring to the programming of websites.Front-end development is the act of developing the user interface and client-side code, while back-end development focuses on the infrastructure behind a website, mainly server-side code.

Since the World Wide Web was released publicly in 1993, web development has evolved greatly, with websites changing from a collection of static HTML pages to complex projects using frameworks, servers, and databases.

OVERVIEW

Web development includes many individual tasks, includingweb design , web content development, networking, and coding.[2] Among web professionals, "web development" usually refers to the main non-design aspects of building websites: writing markup and coding.[3]

India, officially the Republic of India,[j][19] is a country in South Asia.
